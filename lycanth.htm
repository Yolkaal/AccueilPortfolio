Lycanth

Context and project presentation

Lycanth is a Minecraft mod project aimed at introducing fantasy elements into the game. The mod focuses on the addition of werewolves, called "Lycanths", and a new mysterious dimension.

The goal of the project is to enrich the Minecraft experience with new creatures, gameplay mechanics, and lore while keeping the spirit of exploration and adventure.

Main features

• New creature: the Lycanth (werewolf)
• Transformation mechanics
• Unique abilities and gameplay systems
• New dimension to explore
• Original lore connected to the Lycanths

World and lore

The Lycanths are ancient beings tied to a forgotten dimension. Players will discover hidden secrets, mysterious structures, and powerful creatures while exploring this new world.

The project aims to create a coherent lore that integrates naturally into the Minecraft universe.

Technologies used

• Java
• Minecraft Modding API (Forge / Fabric depending on version)
• Game design and worldbuilding
• 3D models and textures
